## 2020.7.9
***
- 解决keep-alive数据缓存问题，看看大家都是怎么做的
- 这个办法有点蠢
``` javascript
beforeRouteEnter(to, from, next) {
    // 解决因为此页面使用了keep-alive导致的非必须的数据缓存问题
    if (from.name === 'luckyCouponList') {
      setTimeout(() => window.location.reload(), 0)
    }
    next()
  },
```
